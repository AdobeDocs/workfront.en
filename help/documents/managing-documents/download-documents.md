# Download Documents

You can download documents from *Adobe Workfront* either individually or in bulk. Documents can be downloaded from any area that contains documents in *Workfront*.&nbsp;

## Download an individual document

1. Go to the project, task, or issue that contains the document, then select `Documents`.
1. Find the document you need.

</ol>

## Download multiple documents simultaneously

You can download multiple documents simultaneously.&nbsp;

Documents linked to *Workfront* from Box, Dropbox, Google Drive, and so forth cannot be downloaded.

<ol> 
 <li value="1">Go to the Documents area containing the documents you want to download.</li> 
 <li value="2">(Optional) Select the individual documents that you want to download from the list of documents.</li> 
 <li value="3">(Optional) Select the folders that contain the documents you want to download from the list of folders.</li> 
 <li value="4"> <draft-comment>
   <p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the Download Selected icon <img src="assets/download-icon.png">.</p>
  </draft-comment><p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the Download Selected icon <img src="assets/download-icon.png">.</p> <p>Folders are downloaded as .zip files and are limited to 4GB.</p> </li> 
</ol>
