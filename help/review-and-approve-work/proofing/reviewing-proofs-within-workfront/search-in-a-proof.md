# Search within a *`proof`* {#search-within-a-proof}

You can quickly locate text in a *`proof`* created for the following types of documents:



* PDF
* Office (.doc, .docx, .odt)
* Static web page




>[!NOTE]
>
>&nbsp;Proofs created prior to April 26, 2017 might not be searchable.

## Search within a *`proof`* {#search-within-a-proof-1}




1. Open the *`proof`* that you want to search.
1.  In the toolbar above the *`proof`*, click the  `Search document`&nbsp;icon.


   ![](assets/search-document-icon-search-in-proof-350x129.png)



1.  Begin typing the text you want to search for.


   The search tool highlights the text in the document as you type.


   ![Search_text_highlighted_as_you_type.png](assets/search-text-highlighted-as-you-type-350x214.png)



1. Finish typing the text you want to search for, then click the  `Up` and  `Down`&nbsp;arrows to scan through the search results within the document.
