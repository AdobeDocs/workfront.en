# Install `Adobe Workfront` for Microsoft Teams

The `Adobe Workfront` for Microsoft Teams app allows you to perform basic actions in `Workfront` without leaving your Microsoft Teams chat channels.

>[!NOTE]
>
>Microsoft Teams no longer supports Internet Explorer. To use the `Adobe Workfront` for Microsoft Teams integration, you must use a web browser other than Internet Explorer.

## Prerequisites

You must be a team owner in Microsoft Teams to install Workfront for Microsoft Teams.

## Install `Workfront` for Microsoft Teams

As a team owner in Microsoft Teams, you can install the `Workfront` for Microsoft Teams app for each of your teams either from the Microsoft Store or from a file provided by `Workfront`.

* [Install Workfront for Microsoft Teams from the Microsoft Store](#installing-from-ms-store) 
* [Install Workfront for Microsoft Team from a private file](#installing-from-a-private-file)

### Install `Workfront` for Microsoft Teams from the Microsoft Store

<ol> 
 <li value="1">Log in to Microsoft Teams as a team owner.</li> 
 <li value="2"> Select the team for whom you want to install the <span>Workfront</span> for Microsoft Teams app.&nbsp; </li> 
 <li value="3"> <p> Click <span class="bold">Store </span>on the side navigation bar.<br></p> <p> <img src="assets/ms-teams-store-icon-319x809.png" alt="ms_teams_Store_icon.png" style="width: 319;height: 809;"> </p> </li> 
 <li value="4"> In the <span class="bold">Search all</span> box, type <em style="font-style: normal; font-weight: bold;">Workfront</em>.<br></li> 
 <li value="5">Click the <span class="bold"><span>Workfront</span></span> card and follow the instructions in the wizard.</li> 
 <li value="6"> <p> (Recommended) Select a team from the <span class="bold">Add to a team</span> drop-down menu and enable the <span class="bold">Yes</span> option to add the app to a team.<br></p> <p> <img src="assets/ms-teams-add-to-a-team-option-350x122.png" alt="ms_teams_add_to_a_team_option.png" style="width: 350;height: 122;"> <br> </p> </li> 
 <li value="7"> <p> Select <span class="bold">General</span> to use the app in that channel for the selected team, then click Set up.<br></p> <p> <img src="assets/teams-select-which-channel-to-use-the-app-install-350x330.png" alt="teams_select_which_channel_to_use_the_app_install.PNG" style="width: 350;height: 330;"> <br> </p> </li> 
 <li value="8"> When the install completes,&nbsp;a notification that the install was successful appears in the General channel of the team you selected. All members of the team can see this notification.</li> 
 <li value="9">(Optional) Pin your <span>Workfront</span> app for easier access: 
  <ol>
   <li value="1">Click the <span class="bold">More</span> icon under the<span class="bold">&nbsp;</span>message&nbsp;field in the General channel.</li>
   <li value="2"><p>Mouse over the <span>Workfront</span> app in the list, then click the <span class="bold">More</span> icon to the right of it.<br></p><p><img src="assets/teams-pin-workfront-link-350x80.png" alt="teams_pin_workfront_link.PNG" style="width: 350;height: 80;"></p></li>
   <li value="3"><p>Click <span class="bold">Pin</span>.<br></p><p>This adds a <span>Workfront</span> icon under the chat field. You can quickly access the Search area from here.<br></p><p>For information about searching for <span>Workfront</span> items, see <a href="../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md" class="MCXref xref">Search for and share Adobe Workfront items in Microsoft Teams</a>.</p></li>
  </ol></li> 
 <li value="10"> <p>Click<span class="bold"> Log in to <span>Workfront</span></span> to access <span>Workfront</span> from Microsoft Teams.<br></p> <p>For information about logging in to <span>Workfront</span>, see the <a href="#logging-in-to-workfront" class="MCXref xref">Log in to Workfront from Microsoft Teams</a> section in this article.&nbsp;</p> </li> 
</ol>

### Install `Workfront` for Microsoft Team from a private file

If your organization restricts the access to downloading apps from the Microsoft Store, you must contact our Support Team and request a private file of the `Workfront` for Microsoft Teams app to install the app.

For information about contacting our Support Team, see [Contact Customer Support](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

To install `Workfront` for Microsoft Teams from a private file:

<ol> 
 <li value="1">Save the private file that you have received from <span>Workfront</span> on your computer.</li> 
 <li value="2">Log in to Microsoft Teams as a Microsoft team owner.</li> 
 <li value="3"> <p>Click the <span class="bold">More</span> icon for the team for which you want to install <span>Workfront</span> for Microsoft Teams.<br></p> <p> <img src="assets/ms-teams-more-icon-for-team-350x177.png" alt="ms_teams_more_icon_for_team.png" style="width: 350;height: 177;"> </p> </li> 
 <li value="4">Click <span class="bold">Manage Team</span>.&nbsp;</li> 
 <li value="5">Select the <span class="bold">Apps</span> tab, then click <span class="bold">Upload a custom app</span> in the lower-right corner of the screen.&nbsp;</li> 
 <li value="6">Browse for the private file that you saved on your computer, then follow the installation steps to install <span>Workfront</span> for Microsoft Teams.</li> 
 <li value="7"> When the install completes, a notification that the install was successful appears in the General channel of the team you selected. All members of the team can see this notification.</li> 
 <li value="8">(Optional) Click the <span class="bold">More</span> icon under the<span class="bold"> Type your questions here</span> field.</li> 
 <li value="9"> <p>(Optional) Mouse over the <span>Workfront</span> app in the list, then click the <span class="bold">More</span> icon to the right of it.<br></p> <p> <img src="assets/teams-pin-workfront-link-350x80.png" alt="teams_pin_workfront_link.PNG" style="width: 350;height: 80;"> </p> </li> 
 <li value="10"> <p>(Optional) Click <span class="bold">Pin</span>.<br></p> <p>This adds a <span>Workfront</span> icon under the Type your questions here field. You can quickly access the Search area from here.<br>For information about searching for <span>Workfront</span> items, see <a href="../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md" class="MCXref xref">Search for and share Adobe Workfront items in Microsoft Teams</a>.</p> </li> 
</ol>

## Log in to `Workfront` from Microsoft Teams

As a Microsoft Teams team owner, you must install the `Workfront` for Microsoft Teams app for your team before you or anyone on the team can log in to `Workfront` from Microsoft Teams.

When you are logged in to `Workfront` from Microsoft Teams, you can receive `Workfront` notifications in the `Workfront` bot channel or you can perform certain actions in `Workfront` from Microsoft Teams.&nbsp;

For information about installing the `Workfront` app, see the [Install Workfront for Microsoft Teams](#installing-workfront-for-ms-teams) section in this article.

For information about accessing `Workfront` from Microsoft Teams to perform certain actions, see [Access Adobe Workfront from Microsoft Teams](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/access-workfront-from-ms-teams.md).

To log in to `Workfront` from Microsoft Teams:

<ol> 
 <li value="1"> <p>Go to the <span class="bold">General</span> channel of the team where the <span>Workfront</span> for Microsoft Teams app has been installed and click<span class="bold"> Log in to <span>Workfront</span></span>.<br></p> <p>The <span>Workfront</span> bot chat channel is added to your Microsoft Teams chat channels.<br></p> <p> <img src="assets/teams-workfront-bot-channel-highlighted-350x220.png" alt="teams_workfront_bot_channel_highlighted.PNG" style="width: 350;height: 220;"> </p> </li> 
 <li value="2"> <p> Go to the <span>Workfront</span> bot chat channel in Microsoft Teams and type <em>log in</em> in the <span class="bold">Type your questions here</span> field.<br></p> <p>Or<br></p> <p>Click <span class="bold">Log in</span>.</p> <p> <img src="assets/ms-teams-log-in-prompt-with-4-tabs-350x139.png" style="width: 350;height: 139;"> <br> </p> <p>A new browser tab opens. </p> </li> 
 <li value="3"> <p>Follow the prompts to log in to <span>Workfront</span> using Enhanced Authentication, OAuth 2.0, or your Security Assertion Markup Language (SAML) URL.</p> <note type="note"> 
   <ul> 
    <li>When you are prompted to enter the <span>domain</span> of your <span>Workfront</span> account, type it using this format: <em>yourCompany'sDomain.my.workfront.com</em>. Your company's domain is usually the name of your company.</li> 
    <li>Enhanced Authentication is not available until a <span>Workfront administrator</span> enables it for this integration. </li> 
   </ul> 
  </note> </li> 
 <li value="4"> <p>Close the browser tab you used to log in and return to Microsoft Teams.<br></p> <p>A notification displays in the <span>Workfront</span> bot chat channel to confirm that you logged in to <span>Workfront</span> successfully.</p> </li> 
</ol>
