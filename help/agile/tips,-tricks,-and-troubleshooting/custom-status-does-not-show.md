# Custom statuses not showing up on the Agile story board {#custom-statuses-not-showing-up-on-the-agile-story-board}

If you want to use a custom status on an agile story board, your `Workfront administrator` must first create that custom status as a system-wide status. For more information, have your `Workfront administrator` see [Create and customize system wide statuses](create-customize-statuses.md).
