# Add or remove members from a board

People must be added to the board as members before they can view the board and be assigned to cards.

## Access requirements

You must have the following access to perform the steps in this article:

<table cellspacing="0"> 
 <col> 
 </col> 
 <col> 
 </col> 
 <tbody> 
  <tr> 
   <td role="rowheader"><span>Adobe Workfront</span> plan*</td> 
   <td> <p>Any</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader"><span>Adobe Workfront</span> license*</td> 
   <td> <p><span>Request</span> or higher</p> </td> 
  </tr> 
 </tbody> 
</table>

&#42;To find out what plan, license type, or access you have, contact your `Workfront administrator`.

## Add members to a board

<ol> 
 <li value="1"> <p>Click the <span class="bold">Main Menu</span> icon <img src="assets/main-menu-icon.png"> in the upper-right corner of <span>Adobe Workfront</span>, then click <span class="bold">Boards</span>.</p> </li> 
 <li value="2"> <p>Create a new board or edit an existing board. For information, see <a href="../../agile/get-started-with-boards/create-edit-board.md" class="MCXref xref">Create or edit a board</a>.</p> </li> 
 <li value="3"> <p>Click the <span class="bold">Add member</span> icon <img src="assets/boards-addmember-spectrum-25x25.png" alt="Add members" style="width: 25;height: 25;">.</p> </li> 
 <li value="4"> <p>In the <span class="bold">Add members</span> box, start typing a name, then select it when it displays in the list.</p> </li> 
</ol>

## Remove members from a board

<ol> 
 <li value="1"> <p>Click the <span class="bold">Main Menu</span> icon <img src="assets/main-menu-icon.png"> in the upper-right corner of <span>Adobe Workfront</span>, then click <span class="bold">Boards</span>.</p> </li> 
 <li value="2"> <p>Create a new board or edit an existing board. For information, see <a href="../../agile/get-started-with-boards/create-edit-board.md" class="MCXref xref">Create or edit a board</a>.</p> </li> 
 <li value="3"> <p>Click the <span class="bold">Add member</span> icon <img src="assets/boards-addmember-spectrum-25x25.png" alt="Add members" style="width: 25;height: 25;">.</p> </li> 
 <li value="4"> <p>In the <span class="bold">Add members</span> box, click the X next to a member name to remove them from the board.</p> <p> <img src="assets/boards-remove-member-from-board-350x367.png" alt="Remove member from board" style="width: 350;height: 367;"> </p> <p>When you delete a member from a board, they are also deleted from any cards they were assigned to.</p> <p>Members are only removed from this board. They are not removed from other boards they belong to.</p> <note type="note">
   You can't remove the board owner.
  </note> </li> 
</ol>
