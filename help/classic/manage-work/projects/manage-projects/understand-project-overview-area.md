# Manage information in the project Overview area

You can view or edit the information of a project by accessing the Overview area of the Project Details tab. There is a limited number of fields that you can view or edit in this area. For information about editing all information for a project see [Edit projects](../../../manage-work/projects/manage-projects/edit-projects.md).

## Access the Overview subtab

1. Go to the project whose Overview subtabyou want to view.
1. Select the Project Details tab. 
1. Select the Overview subtab.Click Edit Overview to edit the project. 
1. (Conditional) If there is a specific field that needs to be updated on a project but is not displayed on this tab , click Edit Project&nbsp; on the project to view more project fields.

   For more information about editing projects, see [Edit projects](../../../manage-work/projects/manage-projects/edit-projects.md).

1. Edit or review the following fields in the Overview subtab. 
   Name: This is the name of the project. Description: Describe the purpose of this project. Reference Number: This is a unique number assigned by Workfront. This field is not editable. URL: Insert any URL in this field. It can be a Workfront URL or any other. Schedule From: Sets how the project is scheduled, e.g., from start date or the completion date. Planned Start Date: When the project is planned to start. This is manually set by the project manager when the project is scheduled from Start Date. Workfront automatically sets this date when the project is scheduled from Completion Date, based on the duration of the tasks in the project. Projected Start Date: This is a 'real time' date of when work is going to start based on completed work and remaining work. This field is not editable. Actual Start Date: The date and time when the work on the project was initiated. It usually coincides with when a task assignee sets the value of a task Status to any other than New for the first time. This field is not editable. Planned Completion Date : When the project is planned to complete. This is manually set by the project manager when the project is scheduled from Completion Date. Workfront automatically sets this date when the project is scheduled from Start Date, based on the duration of the tasks in the project. Projected Completion Date: A ‘real time’ date of when the project is going to be completed based on the progress of completed tasks and based on the progress updates on the tasks that are either New or In Progress status. This field is not editable. Actual Completion Date: The date and time the work is completed. This field is not editable. Planned Duration: Amount of time the project will span. This field is not editable. This field is not editable. Actual Duration: Is the difference between the Actual Start Date and Actual Completion Date of the project. Workfront calculates the Project Duration using the following formula: Actual Duration = WEEKDAYDIFF(Actual Completion Date, Actual Start Date) This field is not editable. Planned Hours: Hours planned on the project. This field is not editable. Actual Hours: Hours actually recorded on the project. This field is not editable. Condition Type: Determines if the manager sets the condition or if Workfront does. Condition: This is the condition of the project. This field is editable only when the Condition Type is set to Manual. Priority: Serves as the designated priority or importance of the project. Status: This is the status of the project. Project Owner: The project manager or day-to-day manager of the project. Project Sponsor: This is the primary stakeholder for the project. This is usually an executive overseeing and championing the project, or it is the person with budgetary responsibility. Group: The Group associated with the project. Company: The company that owns the project. Entered By: Who created the project. Entry Date: The date the project was created. This field is not editable. Last Updated By: The user who created the last update. This field is not editable. Last Update Date: The date the last changes to the project details were made. This field is not editable. 
1. Click Save.
