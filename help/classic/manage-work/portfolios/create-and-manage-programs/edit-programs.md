# Edit programs

You can edit information on programs that you have created, or that other users have created if they shared them with you.

You can edit a program in the program page or you can edit programs in a list.

Edit programs Go to the Projects area in the Global Navigation Bar. Select the Portfolios tab, click the name of a portfolio, then click the Programs tab. Do one of the following: Click the name of a program to open it, then click Program Details. Select one or more programs in the list, then click Edit to open the Edit Program box. Consider updating the following fields in the Program Details tab or the Edit Program box: Name Update the name of the program. Tip: This is not available in the Edit Program box when you select more than one program. Description Type a description for the program to indicate what is unique about it. Program Manager Start typing the name of a user that you want to indicate as the program manager, then select it when it appears in the list. This is the person who can oversee the work defined in the projects of the program. Important: When you designate someone as the Program Manager, they automatically gain Manage permissions to the program and the projects in the program. Group Add the name of a single group if the group is associated with the program or has responsibility for completing it. Is Active Select this checkbox if you want the program to be active. Other users can find active programs and attach them to projects or add them to portfolios. Inactive programs cannot be attached to projects or portfolios. This is enabled by default. (Optional) Click Custom Forms. (Optional) Select the custom form or forms that you want to associate with the program. You must build the custom forms before they are available to select in this field. Only active custom forms display in the list. For more information about building custom forms, see Create or edit a custom form. You can add up to ten custom forms to a program. (Conditional) If editing the program in the Program Details box, click Edit Custom Form to update the fields on a custom form. Note: Depending on how your Workfront administrator set the permissions for the sections in your custom form, not everyone can view or edit the same fields on a given custom form. The permissions to edit fields within a section of a custom form depend on the permissions you have on the program itself. For information about setting permissions on sections of a custom form, see Create or edit a custom form. (Conditional) In the Edit Program box, click Comment. Do the following: Add a comment in the Post an update to the program field. (Optional) Click the People icon to add a user or a team to the comment. (Optional) Click the Lock icon to lock the comment and make it private to only users in your company. After you save your changes, the comment you add displays in the program's Updates tab and sends an email to the users included in it. Tip: You cannot add a comment in the Program Details tab. (Conditional) Click Save if you edit the program in the Program Details tab Or Click Save Changes if you edit the program in the Edit Program box. (Optional) From the program page, click the Projects tab to add projects to the program. For more information, see Add a project to a program . Edit programs in a list Go to a list of programs and select one or more programs that you want to edit, then click the Edit icon at the top of the list. The Edit Programdialog box displays. All program fields are available in the Edit Program box and are grouped by the areas listed in the left panel. Consider specifying information in any of the following sections: Overview Custom Forms Comment Overview Begin editing a program as described above. Click Overview and specify the following fields: Name Update the name of the program. Tip: This is not available when you selected more than one program. Description Program Manager Group Add the name of a single group if the group is associated with the program or has responsibility for completing it. Is Active Click Save Changes or continue editing the following sections. Custom Forms Begin editing the program as described above. Click the Add Forms drop-down menu to select a custom forms and add it to the program. You must create program custom forms before they are available to add. Note: Depending on how your Workfront administrator set the permissions for the sections in your custom form, not everyone can view or edit the same fields on a given custom form. The permissions to edit fields within a section of a custom form depend on the permissions you have on the program itself. For information about setting permissions on sections of a custom form, see Create or edit a custom form. Update any fields in the custom forms, then click Save Changes or continue with the following section. Comment Begin editing a program as described above. Click Comment. Add a comment in the Post an update to the program field. (Optional) Click the People icon to add a user or a team to the comment. (Optional) Click the Lock icon to lock the comment and make it private to only users in your company.
