## View user personal time off {#view-user-personal-time-off-1}




1.  `<MadCap:conditionalText data-mc-conditions="QuicksilverOrClassic.Quicksilver"> Click the  <span class="bold">Main Menu</span> icon  <img src="assets/main-menu-icon.png"> in the upper-right corner, then click&nbsp; <span class="bold">Reports >&nbsp;New Report</span>.</MadCap:conditionalText>` 
1.  From the drop-down menu, select `Time Off`.
1.  Click `Save + Close`.


   The report displays the following fields in the view by default:


1. (Optional) Finish creating the report by editing any of the following tabs:
    
    
    * Columns (View)
    * Groupings
    * Filters
    * Chart
    
    
   For information about creating reports, see the article [Create a custom report](create-custom-report.md).


   >[!TIP] {type="tip"}
   >
   >We recommend adding a grouping for the User object, to make the report easier to read.
