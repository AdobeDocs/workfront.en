## Add or edit a visualization in a report

<ol> 
 <li value="1"> <p>Click the <span class="bold">Main Menu</span> icon <img src="assets/main-menu-icon.png"> in the upper-right corner of <em>Adobe Workfront</em>, then click <span class="bold">Reports</span>.</p> </li> 
 <li value="2"> <p>Click <span class="bold">New report</span>.</p> <p>Or</p> <p>Go to an existing report, click the <span class="bold">More Menu</span> icon <img src="assets/more-icon.png"> in the report header, then select <span class="bold">Edit</span>.</p> </li> 
 <li value="3">(Conditional) In the report building that displays, if there isn't already a table on the canvas, add and configure one as explained in <a href="../../reports-and-dashboards/new-reporting-experience/add-or-edit-report-table.md" class="MCXref xref">Add or edit a report table in Reporting Canvas</a>.</li> 
 <li value="4"> <p>From the right panel, double-click the <span class="bold">Visualization</span> icon <img src="assets/visualization-icon.png"> or drag it onto the canvas where you want it.</p> <p>If you double-click the icon, the visualization displays at the top of the canvas in its default size, which you can change by dragging its corner handles.</p> </li> 
 <li value="5"> <p>Select the table (listed on the visualization block) that you want the visualization to depict.</p> <p> <img src="assets/select-table-on-vis-350x155.png" style="width: 350;height: 155;"> </p> </li> 
 <li value="6"> <p>Select the type of visualization you want to add.</p> <p> <img src="assets/select-vis-type-350x205.png" style="width: 350;height: 205;"> </p> <p>
