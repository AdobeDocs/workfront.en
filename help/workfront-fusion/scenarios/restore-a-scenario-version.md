## Restore a scenario version

<ol> 
 <li value="1">Open your scenario.</li> 
 <li value="2">Click the More icon <img src="assets/more-icon.png">, then click <span class="bold">Previous Versions</span>.</li> 
 <li value="3">Click the <span class="bold">Pick the version</span>, then click the version you would like to restore.</li> 
 <li value="4"> <p>Click <span class="bold">OK</span>:</p> </li> 
 <li value="5">The selected scenario version will be restored.</li> 
</ol>

>[!NOTE]
>
>The restored version of the scenario is not automatically saved. If you want to save the restored version of the scenario, you have to do so manually.
