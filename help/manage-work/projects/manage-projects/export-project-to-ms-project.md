# Export a project to Microsoft Project  {#export-a-project-to-microsoft-project}

You can export *`Adobe Workfront`* projects to Microsoft Project.&nbsp;




>[!IMPORTANT] {type="important"}
>
>Not all *`Workfront`* fields are transferred in the Microsoft Project file.  
>For more information about the compatibility of fields between *`Workfront`* and Microsoft Project, see the article [Map Microsoft Project fields to Adobe Workfront](map-ms-project-fields-to-workfront.md).





We recommend that you limit the number of times you transfer projects from one application to another.&nbsp;

To export a project from *`Workfront`* to Microsoft Project:



1. Go to the project you want to export.
1.  `<MadCap:conditionalText data-mc-conditions="QuicksilverOrClassic.Quicksilver"> Click the  <span class="bold">More</span> icon  <img src="assets/qs-more-menu.png">, then click&nbsp; <span class="bold">Export MS&nbsp;Project</span>. </MadCap:conditionalText>`   
  
   The project is downloaded as an XML file to your computer and it is ready to be imported into Microsoft Project.&nbsp;
