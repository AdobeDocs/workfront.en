# Create template from project

You can create templates when you save an existing project as a template.

After you save an existing project as a template, you can use the new template to create new projects. This simplifies and expedites the project creation process.

## Access requirements

You must have the following access to perform the steps in this article:

 <col> 
 <col> 
 <tbody>

## Create template from project

<ol> 
 <li value="1">Go to the project that you want to save as a template. </li> 
 <li value="2"> <draft-comment>
   <p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">More</span> menu <img src="assets/qs-more-icon-on-an-object.png">, then <span class="bold">Save as Template</span>. </p>
  </draft-comment><p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">More</span> menu <img src="assets/qs-more-icon-on-an-object.png">, then <span class="bold">Save as Template</span>. </p> </li> 
 <li value="3"> <p>Specify the following information for the template:</p> 
  <table cellspacing="0"> 
   <col> 
   <col> 
   <tbody> 
    <tr> 
     <td role="rowheader">Name</td> 
     <td>Specify a name for the template.</td> 
    </tr> 
    <tr> 
     <td role="rowheader">Description</td> 
     <td>Provide a description for the template.</td> 
    </tr> 
    <tr> 
     <td role="rowheader">Is Active</td> 
     <td> <p>Select from the following options:</p> 
      <ul> 
       <li> <p><span class="bold">Yes</span>: Other users can find the template and attach it to projects.</p> </li> 
       <li><span class="bold">No</span>: Other users cannot find the template and cannot attach it to projects.</li> 
      </ul> </td> 
    </tr> 
    <tr> 
     <td role="rowheader">Custom Forms</td> 
     <td>Use the drop-down list to select any custom forms to attach to the template. If any custom forms have already been associated with the project, all of the data fields from those custom forms are displayed.<br>You can include up to 10 custom forms on a single template.</td> 
    </tr> 
   </tbody> 
  </table> </li> 
 <li value="4">Click <span class="bold">Manage Forms</span> to remove or reorder the forms. For information about how to remove and reorder custom forms on the template, see <a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-and-manage-custom-forms.md" class="MCXref xref">Custom forms</a>.<p><img src="assets/save-as-template-first-step-350x167.png" style="width: 350;height: 167;"></p></li> 
 <li value="5">Click <span class="bold">Next Step.</span></li> 
 <li value="6"> <p>In the<span class="bold"> Options</span> section, select the checkbox beside any information you want to clear from the template. </p> <p> <img src="assets/save-as-template-options-step-350x147.png" alt="" style="width: 350;height: 147;"> <br> </p> </li> 
 <li value="7">Click <span class="bold">Next Step.</span></li> 
 <li value="8"> <p>In the <span class="bold">Exclude</span> section, select any tasks that you want to exclude from the project.<br></p> <p> <img src="assets/save-as-template-exclude-350x205.png" style="width: 350;height: 205;"> </p> </li> 
 <li value="9"> <p>Click <span class="bold">Finish and Save Template.</span></p> <p>Your template now appears in the list of available templates and can either be attached to an existing project or used to create a new one. </p> </li> 
</ol>
