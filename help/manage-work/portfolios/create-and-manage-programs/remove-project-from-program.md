# Remove a project from a program  {#remove-a-project-from-a-program}

You may remove a project from a program and do one of the following:



*  Move it to another program.
*  Permanently remove it from the current program and do not associate it with another program.
*  Permanently delete the project.


  For information about deleting projects, see [Delete projects](delete-projects.md).

## Remove a project from a Program {#remove-a-project-from-a-program-1}




1.  `<MadCap:conditionalText data-mc-conditions="QuicksilverOrClassic.Quicksilver"> Go to a program and click  <span class="bold">Projects</span> in the left panel. </MadCap:conditionalText>`   

1. Select the project you want to remove
1. Click the `Remove Projects`&nbsp;drop-down menu.
1.  Click `Delete` >  `Yes, Delete it` to delete the project from *`Workfront`*. 


   or 


   Click `Remove` `from Program` to remove it from the selected program.  



   If you clicked Remove from Program, the project is no longer associated with the selected program.
