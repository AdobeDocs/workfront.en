# Update or edit a work item in the Home area

You can view and add updates to a work item within the Home area in *Adobe Workfront*. You can also edit other data related to the work item if you have access to edit it.

## Access requirements

You must have the following access to perform the steps in this article:

You can view updates on any work item in the Work List:

## Provide updates on a work item

You can provide updates on any work item in the Work List:

<ol> 
 <li value="1"> <draft-comment>
   <p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">Main Menu</span> <img src="assets/main-menu-icon.png"> in the upper-right corner, then click&nbsp;<span class="bold">Home</span>. </p>
  </draft-comment><p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">Main Menu</span> <img src="assets/main-menu-icon.png"> in the upper-right corner, then click&nbsp;<span class="bold">Home</span>. </p> </li> 
 <li value="2">In the <span class="bold">Work List</span> area, select the item where you want to provide an update.</li> 
 <li value="3"> <p>In the right panel, click <span class="bold">Update</span> to display a text field.<br><draft-comment>
    <img src="assets/updating-a-work-item-update-stream-350x295.png" style="width: 350;height: 295;" data-mc-conditions="QuicksilverOrClassic.Draft mode">
   </draft-comment><img src="assets/updating-a-work-item-update-stream-350x295.png" style="width: 350;height: 295;" data-mc-conditions="QuicksilverOrClassic.Draft mode"></p> <draft-comment>
   <p data-mc-conditions="QuicksilverOrClassic.Quicksilver"> <img src="assets/make-an-update-box-expanded-home-nwe-350x204.png" style="width: 350;height: 204;"> <br> </p>
  </draft-comment><p data-mc-conditions="QuicksilverOrClassic.Quicksilver"> <img src="assets/make-an-update-box-expanded-home-nwe-350x204.png" style="width: 350;height: 204;"> <br> </p> </li> 
 <li value="4">In the provided field, specify your update.</li> 
 <li value="5">(Optional) Specify any of the following information:<br><span class="bold">Commit Date:</span> Select a date from the date picker when you commit to complete the work item.<br><span class="bold">How's it going?:</span>&nbsp;Select a new condition for the task or issue. For more information about task and issue conditions, see <a href="../../../manage-work/projects/updating-work-in-a-project/update-condition-for-tasks-and-issues.md" class="MCXref xref">Update Condition for tasks and issues</a>.<br><span class="bold">Status:</span>&nbsp;Select a new status for the task or issue. You can also double-click the completion bar and enter the percent complete. </li> 
 <li value="6">Click <span class="bold">Update</span>.</li> 
</ol>

## Edit a work item

You can edit any field on any work item that you have access to edit.

<ol> 
 <li value="1"> <draft-comment>
   <p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">Main Menu</span> <img src="assets/main-menu-icon.png"> in the upper-right corner, then click&nbsp;<span class="bold">Home</span>. </p>
  </draft-comment><p data-mc-conditions="QuicksilverOrClassic.Quicksilver">Click the <span class="bold">Main Menu</span> <img src="assets/main-menu-icon.png"> in the upper-right corner, then click&nbsp;<span class="bold">Home</span>. </p> </li> 
 <li value="2">In the <span class="bold">Work List</span> area, select the item you want to edit.<br>All fields you have access to view are displayed in the right panel.</li> 
 <li value="3"> <p>In the right panel, hover over the field you want to edit.<br>If you have rights to edit the field, the field is highlighted as you hover over it.<br></p> <p> <br> <draft-comment>
    <img src="assets/home-350x90.png" style="width: 350;height: 90;" data-mc-conditions="QuicksilverOrClassic.Quicksilver">
   </draft-comment><img src="assets/home-350x90.png" style="width: 350;height: 90;" data-mc-conditions="QuicksilverOrClassic.Quicksilver"> </p> </li> 
 <li value="4">Click the field you want to edit, then make your desired update.<br></li> 
 <li value="5">Click away from the field to save your changes.</li> 
</ol>
