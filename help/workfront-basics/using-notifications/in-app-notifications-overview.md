# In-app notifications overview {#in-app-notifications-overview}

In-app notifications are available both from the web application and the mobile application.&nbsp;They&nbsp;keep you informed about two types of information:



* Announcements from your *`Adobe Workfront administrator`*
* Activities regarding your work




>[!NOTE]
>
>In-app notifications for work activities are not tied to email notifications in *`Workfront`*. For more information, see [Adobe Workfront notifications](wf-notifications.md).




## In-app notification types {#in-app-notification-types}




* [Announcement notification type](#announce) 
* [Work item notification types](#work2)

*  
    
    
    * Assignment notifications are sent only when the project status is set to Current (or to a custom status that is the equivalent of Current).
    * It is&nbsp;not intended for Requestors and Reviewers to be assigned work. Therefore, they are not notified if they are assigned to tasks and issues.&nbsp;
    * If you assign work to yourself or a team you belong to, you will not receive a notification.
