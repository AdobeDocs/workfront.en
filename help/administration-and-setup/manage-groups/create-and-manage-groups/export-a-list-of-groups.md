# Export a list of groups {#export-a-list-of-groups}

When you are viewing groups that you manage, you can export a list of those groups.


If there are any groups above the group you manage, their administrators can also do this for your group. The same is true for *`Workfront administrators`* (for any group).

&#42;If you need to find out what plan or license type you have, contact your *`Workfront administrator`*.



1. Click the `Main Menu` icon ![](assets/main-menu-icon.png) in the upper-right corner of *`Adobe Workfront`*, then click `Setup` ![](assets/gear-icon-settings.png).

1.  In the left panel, click `Groups` ![](assets/groups-icon.png).


   In the list of groups that displays, *`group administrators`* can see the groups they manage, as well as any subgroups of those groups. *`Adobe Workfront administrators`* can see all groups.

1.  If you want to export the entire group of groups that you manage, click `<MadCap:conditionalText data-mc-conditions="QuicksilverOrClassic.Quicksilver"> the Export icon  <img src="assets/export.png"></MadCap:conditionalText>`. 


   Or


   To export some of the groups you manage, select those groups, then click `<MadCap:conditionalText data-mc-conditions="QuicksilverOrClassic.Quicksilver"> the Export icon  <img src="assets/export.png"></MadCap:conditionalText>`. 

1. Select the file format you want for the exported list.
