

# Edit custom forms and custom fields in a list

When you view a list of the custom forms and custom fields in the system, options in the toolbar let you edit them, limit who they are shared with, copy them, and delete them. You can also export the list of custom forms.

## Edit and limit sharing for custom forms and custom fields

<ol> 
 <li value="1">Click the <span class="bold">Main Menu</span> icon <img src="assets/main-menu-icon.png"> in the upper-right corner of <em>Adobe Workfront</em>, then click <span class="bold">Setup</span> <img src="assets/gear-icon-settings.png">.</li> 
 <li value="2"> <p>Click <span class="bold">Custom Forms</span>.</p> <p>With the <span class="bold">Forms</span> tab selected, all of your organization's custom forms are listed, along with their descriptions, the object type associated with each custom form (custom forms are specific to object type), the name of the user who created the custom form, and whether it is active.</p> <p>For information about what you can do when you select a custom form in the list, see the following articles:</p> 
  <ul> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-or-edit-a-custom-form.md" class="MCXref xref">Create or edit a custom form</a> </li> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/share-access-to-a-custom-form.md" class="MCXref xref">Share a custom form</a> </li> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/copy-custom-form-to-create-a-new-one.md" class="MCXref xref">Copy a custom form to create a new one</a> </li> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/delete-or-deactivate-a-custom-form.md" class="MCXref xref">Delete or deactivate a custom form</a> <br> </li> 
  </ul> </li> 
 <li value="3"> <p>Click the <span class="bold">Fields</span> tab.</p> <p>All of your organization's Custom Fields display in the Field List view, along with information about the type of each field, instructions and details about the field, and a list of the custom forms that contain it.</p> <p>The list contains only fields that meet at least one of the following criteria:</p> <p> 
   <ul> 
    <li>Editable system-wide</li> 
    <li>Visible system-wide</li> 
    <li>Created by you</li> 
    <li>Shared with you, with either View or Manage access</li> 
   </ul> </p> <note type="note">
    Custom Fields that are Section Breaks are not displayed on the Fields tab.
  </note> <p>For information about what you can do when you select a custom field in the list, see the following articles:</p> 
  <ul> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/edit-a-custom-field.md" class="MCXref xref">Edit a custom field</a> </li> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/configure-sharing-for-a-custom-field.md" class="MCXref xref">Configure sharing for a custom field</a> </li> 
   <li><a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/delete-a-custom-field.md" class="MCXref xref">Delete a custom field</a> </li> 
  </ul> </li> 
 <li value="4"> <p>Select the <span class="bold">Sections</span> tab.</p> <p>All the Custom Fields that are Section Breaks display in the Parameter Group List view, along with a list of custom forms that contain the Section Break.</p> <p>For information about editing a custom section that you select in the list, see .</p> </li> 
</ol>
