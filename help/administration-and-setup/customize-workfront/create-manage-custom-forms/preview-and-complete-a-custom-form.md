# Preview and complete a custom form

The highlighted information on this page refers to functionality available only in the new Workfront experience beta.

When you are finished building or editing a custom form, you can preview it to see how it will look to users, then save and close it.

## Preview and finish creating a custom form

<ol> 
 <li value="1">Begin creating and configuring a custom form, as described in the articles <a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-or-edit-a-custom-form.md" class="MCXref xref">Create or edit a custom form</a>, <a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/add-a-custom-field-to-a-custom-form.md" class="MCXref xref">Add a custom field to a custom form</a>, <a href="../../../administration-and-setup/customize-workfront/create-manage-custom-forms/add-image-or-edit-its-properties-in-a-custom-form.md" class="MCXref xref">Add an image or edit its properties in a custom form</a>,and others.</li> 
 <li value="2">(Optional) Click <span class="bold">Preview</span> to see how the form will look when being used, then click <span class="bold">End Preview</span> to return to editing the form.<br></li> 
 <li value="3"> <p>Click <span class="bold">Save+Close</span> to save your changes.<br></p> <p>Or<br></p> <p>If you don't want to keep your changes on the form, click <span class="bold">Cancel</span>, then click <span class="bold">Go Ahead and Close</span>.</p> <note type="note">
   <span class="preview">If the form is already attached to a lot of objects, it might take a few minutes for your changes to apply to all of them. This happens in the background, so you can work on other tasks in <em>Workfront</em> while you wait.</span> 
   <br> 
  </note> </li> 
 <li value="4"> <p>(Optional) Add the form to the <em>Workfront</em> object where you want to use it, as described in <a href="../../../workfront-basics/work-with-custom-forms/add-a-custom-form-to-an-object.md" class="MCXref xref">Add a custom form to an object</a>.</p> <p>Regular users can also do this if they have manage rights to the object.</p> <p>After you add the custom form to an object, users with permissions to manage the object and edit the custom form can edit the information in the custom fields.</p> </li> 
</ol>
