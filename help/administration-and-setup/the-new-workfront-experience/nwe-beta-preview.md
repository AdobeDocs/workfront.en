



# The new Workfront experience Beta Preview {#the-new-workfront-experience-beta-preview}

The new Workfront experience provides a fresh, modern design in addition to new functionality. This page describes changes that were made available in the Preview environment from July 18, 2019 through September 12, 2019.


For the most recent enhancements in the new Workfront experience, see [The new Workfront experience: Beta release activity](nwe-beta-preview-release-activity.md).
