



# The new Workfront experience Beta release activity 2019 {#the-new-workfront-experience-beta-release-activity}

The new Workfront experience provides a fresh, modern design in addition to new functionality. This page describes changes that were made available in the Preview and Production environment from September 12, 2019 to December 31, 2019. For changes made only to the Preview environment prior to these dates, see [The new Workfront experience Beta Preview](nwe-beta-preview.md).


For the most recent enhancements in the new Workfront experience, see [The new Workfront experience: Beta release activity](nwe-beta-preview-release-activity.md).
