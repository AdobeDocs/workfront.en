# Add stories and issues from the [!UICONTROL Scrum] board

You can quickly create a new story or issue directly from the Scrum board, or add an existing story or issue from the board.

>[!NOTE]
>
>The work item [!UICONTROL Planned Start Date] and [!UICONTROL Planned Completion Date] are affected by a setting on the [!UICONTROL Edit Team] page. For information, see the section [[!UICONTROL Configure] how dates are applied when adding work items to an iteration](../../../agile/get-started-with-agile-in-workfront/configure-scrum.md#configur5) in the article [Configure Scrum](../../../agile/get-started-with-agile-in-workfront/configure-scrum.md).

## Create a new story or issue from the Scrum board

{{step1-to-team}}

1. (Optional) Click the **[!UICONTROL Switch team]** icon ![Switch team icon](assets/switch-team-icon.png), then either select a new Scrum team from the drop-down menu or search for a team in the search bar.
1. In the left panel, select **[!UICONTROL Iterations]** to choose a specific iteration, or select **[!UICONTROL Current Iteration]**.
1. Click **[!UICONTROL Add]** on the right side of the Scrum board and select **[!UICONTROL New Story]** or **[!UICONTROL New Issue]**.
1. In the **[!UICONTROL New]** dialog box, add the following information:

   <table style="table-layout:auto">
    <tr>
        <td><strong>[!UICONTROL Name]</strong></td>
        <td>(Required) The name of the story or issue.</td>
    </tr>
    <tr>
        <td><strong>[!UICONTROL Description]</strong></td>
        <td>A description of the item.</td>
    </tr>
    <tr>
        <td><strong>[!UICONTROL Estimate]</strong></td>
        <td>The estimated number of hours or points for the item.</td>
    </tr>
    <tr>
        <td><strong>[!UICONTROL Parent Project]</strong></td>
        <td>(Required) The project where the new story or issue is stored. Start typing the name of a project, then select it when it displays in the list.</td>
    </tr>
   </table>

1. Click **[!UICONTROL Add Story]** or **[!UICONTROL Add Issue]**.

## Add an existing story or issue from the [!UICONTROL Scrum] board

{{step1-to-team}}

1. (Optional) Click the **[!UICONTROL Switch team]** icon ![Switch team icon](assets/switch-team-icon.png), then either select a new Scrum team from the drop-down menu or search for a team in the search bar.
1. In the left panel, select **[!UICONTROL Iterations]** to choose a specific iteration, or select **[!UICONTROL Current Iteration]**.
1. Click **[!UICONTROL Add]** on the right side of the Scrum board and select **[!UICONTROL Existing Story]** or **[!UICONTROL Existing Issue]**.
1. In the **[!UICONTROL Add Existing from Backlog]** dialog box, start typing the name of a story or issue, then select it when it displays in the list.
1. Click **[!UICONTROL Add Story]** or **[!UICONTROL Add Issue]**.
