# Move documents

A user with manage rights on a document can move the document to another object.

The user must&nbsp;also have permissions to add documents to the new object to complete this action.&nbsp;

When you move a document, any of the following will also move with the document:

* Document versions
* Document proofs
* Document approvals

## Move a document

1. Go to the project, task, or issue that contains the document, then select **Documents**.
1. Find the document you need.

1. Click the **Move** icon ![Move icon](assets/move-document--25x19.png).
1. From the drop-down menu in the box that appears, click **Issue**, **Project** or **Task** to indicate the type of object where you want to move the document.&nbsp;

1. Type the name of the **Issue**, **Project** or **Task** in the text box.

1. Click **Finish**.

You can also move a document from the Document Details page.
