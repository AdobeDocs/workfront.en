# View authentication logs in [!DNL Workfront] Proof

>[!IMPORTANT]
>
>This article refers to functionality in the standalone product [!DNL Workfront] Proof. For information on proofing inside [!DNL Adobe Workfront], see [Proofing](../../../review-and-approve-work/proofing/proofing.md).

As a [!DNL Workfront] Proof administrator, you can view the usage information for every user that has logged into [!DNL Workfront] Proof.

1. In the left navigation panel, click **Activity**.
1. In the **Proofs and media logs**&nbsp;drop-down menu, click **Authentication logs**.\
   The login history for every user in the system displays.
